
Use `GParted` if you like GUI.

### `fdisk`

A partition tool, focus on MBR

```bash
fdisk -l        # list
fdisk /dev/sda1 # enter interactive interface
```

In the interactive interface

```bash
m # help
n # new partition
p # print partition table
t # change type
w # write to disk, all operations before 'w' are stored in memory only
# fdisk gives detailed explanations
```
